《AI 生成代码小白急救包》

这是一套给编程小白准备的资料包。 它不是教你成为程序员，也不承诺让你马上赚钱。 它只解决一个很具体的问题：

AI 已经把代码写出来了，但你不知道怎么检查、怎么运行、怎么排错、怎么部署。

Prompt 包

教你怎么问 AI，让 AI 一步一步解释代码、运行方式和报错原因。

运行检查清单

拿到 AI 代码后，按照清单检查文件、命令、环境、依赖和 API Key。

部署教程

教你把简单网页上传到 GitHub Pages，把前端项目部署成在线链接。

常见报错手册

把常见英文报错翻译成人话，并给出可以复制给 AI 的提问模板。

适合谁？

  • 刚开始用 ChatGPT / Cursor 写代码的人
  • 完全不懂代码，但想做一个网页或小工具的人
  • AI 生成代码后，不知道怎么运行的人
  • 经常遇到报错，但不知道怎么问 AI 的人
  • 想把自己的第一个网页部署上线的人

不适合谁？

  • 已经很懂编程的人
  • 想学习完整计算机课程的人
  • 想做复杂商业系统、支付系统、大型后台的人
  • 想靠一份资料马上赚钱的人
  • 完全不愿意动手操作的人
提醒：AI 生成代码只是草稿，重要项目上线前一定要测试。不要把 API Key、密码、Token 上传到公开 GitHub。
